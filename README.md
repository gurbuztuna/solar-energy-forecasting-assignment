# Solar Energy Forecasting Assignment for my Fundamentals of Data Analytics Lesson 

This repository contains the Jupyter Notebook implementation of a data analytics assignment focused on solar energy production forecasting using machine learning models.

## Contents
- Data cleaning and exploratory data analysis
- Model development (Linear Regression and Random Forest)
- Model evaluation
- Energy production forecasting for January 2026
